1. Home Screen (Campus Service)
This is the main navigation page.

College Header: The logo and name "K.S. RANGASAMY COLLEGE OF TECHNOLOGY" are visible.

Main Navigation Buttons: The central part of the screen offers three key services:

RAISE A COMPLAINT

LOST AND FOUND

STUDENT FEEDBACK

2. Login / Raise a Complaint Screen
This screen is titled "RAISE A COMPLAINT" (the 'login' title above it may indicate this is the first action after logging in, or the overall section).

Purpose: To submit a formal complaint.

Input Fields: It collects standard complaint details:

STUDENT NAME

REGISTER NO (Registration Number)

EMAIL ID

STUDENT DEPARTMENT (Dropdown)

COMPLAINT TYPE (Dropdown)

COMPLAINT (Text Area for details)

DATE OF INCIDENT

Action: A SUBMIT button.

3. Lost and Found Screen
This screen is titled "VIRTUAL LOST & FOUND".

Purpose: To facilitate the reporting and searching of lost or found items.

Input Fields: It captures details about the item and the person reporting it:

STUDENT NAME

REGISTER NUMBER

INTERCOM/EXT. (Extension number)

TYPE OF OBJECT (Dropdown)

ROOM NUMBER

CONTACT NUMBER

ITEM DESCRIPTION (Text Area)

STUDENT/STAFF (Dropdown)

LOST OR FOUND (Dropdown to specify the action)

Actions:

SAVE LOST/FOUND (To submit a new report)

VIEW LOST/FOUND (To view existing reports)

4. Feedback Screen
This screen is titled "STUDENT'S SUBJECT FEEDBACK".

Purpose: To collect feedback from students specifically on their academic subjects.

Input Fields:

STUDENT NAME

REGISTER NO

SEMESTER (Dropdown)

YEAR (Dropdown)

STUDENT DEPARTMENT (Dropdown)

SUBJECT CODE (Dropdown)

Action: A SUBMIT FORM button (which would presumably lead to the actual feedback questions).
